## Teaching Notes

This notebook is intended as the first write-it-yourself sequence task after students have practised modify and debug activities. It deliberately revisits exact output, variable naming, `input()`, arithmetic, and `int()`/`float()`/`str()` conversions without introducing any later constructs.

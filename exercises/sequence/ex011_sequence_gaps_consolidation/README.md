# Sequence Gaps Consolidation

**ID**: ex011
**Construct**: sequence
**Type**: gaps
**Difficulty**: medium

## Learning Objective
Consolidate the sequence strand by filling in missing lines that use print output, variables, input, arithmetic, and f-strings.

## Files
- Notebook: [notebooks/student.ipynb](notebooks/student.ipynb)
- Solution: [notebooks/solution.ipynb](notebooks/solution.ipynb)
- Tests: [tests/test_ex011_sequence_gaps_consolidation.py](tests/test_ex011_sequence_gaps_consolidation.py)

## Summary
This is a 10-part gap-fill consolidation exercise. Parts 1-3 use single-line gaps; parts 4-6 use short multi-line gaps; parts 7-10 combine earlier sequence ideas, including f-strings, in larger prompt-and-output tasks.

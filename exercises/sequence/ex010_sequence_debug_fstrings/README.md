## Usage summary

Students open `notebooks/student.ipynb`, fix each tagged `exerciseN` cell, and then complete the matching `explanationN` reflection cell.

From the repository root, teachers can run:

`uv run pytest -q exercises/sequence/ex010_sequence_debug_fstrings/tests/test_ex010_sequence_debug_fstrings.py`

and verify the reference notebook with:

`uv run python scripts/run_pytest_variant.py --variant solution exercises/sequence/ex010_sequence_debug_fstrings/tests/test_ex010_sequence_debug_fstrings.py -q`

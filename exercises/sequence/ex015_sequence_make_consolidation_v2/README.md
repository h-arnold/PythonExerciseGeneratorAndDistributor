## Teaching Notes

This notebook is intended as the final sequence exercise before moving to the selection construct. It consolidates every concept taught in ex002–ex014 by asking students to write complete programs from scratch.

The 10 exercises follow a carefully designed scaffolding ramp:

- **Exercises 1–2**: Highly scaffolded — exact variable names and formulas are provided.
- **Exercises 3–4**: Some scaffolding — task is described with formulas but variable names are left to the student.
- **Exercises 5–6**: Less scaffolding — only a brief description of inputs and expected output.
- **Exercises 7–8**: Minimal scaffolding — just a problem description and the expected output.
- **Exercises 9–10**: Little to no scaffolding — the student must design the entire solution from just a real-world scenario.

See `OVERVIEW.md` for pacing, misconceptions, and suggested classroom delivery.

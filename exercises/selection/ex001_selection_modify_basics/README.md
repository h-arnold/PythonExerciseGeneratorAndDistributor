# Selection Modify Basics — ex001

## Learning Objective
Modify existing programs that use simple `if` statements with `==`, `!=`, `<`, and `>` comparison operators.

## Notebook Path
- Student: `notebooks/student.ipynb`
- Solution: `notebooks/solution.ipynb`

## Tests Path
- Canonical tests: `tests/test_ex001_selection_modify_basics.py`

## Summary
Ten modification exercises where students change working code to meet new requirements. Starting with a simple static font-checking program, students progress through temperature thresholds, not-equal checks, age gates, quiz answers, score thresholds, name checks, height restrictions, drink orders, and finally a password check. All exercises use only `if` statements (no `elif` or `else`) and the four comparison operators `==`, `!=`, `<`, `>`.

## Teacher notes
- Created: 2026-06-16
- Target concepts: `if` statements, comparison operators (`==`, `!=`, `<`, `>`), modifying condition values and output messages
- Prerequisites: Sequence construct (variables, `print()`, `input()`, `int()` casting)
- Exercise type: Modify (students edit working code)

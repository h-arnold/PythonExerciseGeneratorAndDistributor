# Selection Modify Basics — ex001

## Learning Objective
Modify existing programs that use `if` statements (exercises 1–5) and `if-else` logic (exercises 6–10) with `==`, `!=`, `<`, and `>` comparison operators.

## Notebook Path
- Student: `notebooks/student.ipynb`
- Solution: `notebooks/solution.ipynb`

## Tests Path
- Canonical tests: `tests/test_ex001_selection_modify_basics.py`

## Summary
Ten modification exercises where students change working code to meet new requirements. Starting with a simple static font-checking program, students progress through temperature thresholds, not-equal checks, age gates, quiz answers, score thresholds, name checks, height restrictions, drink orders, and finally a password check. Exercises 1–5 use simple `if` statements with the four comparison operators `==`, `!=`, `<`, `>`. Exercises 6–10 introduce `if-else` logic, so students modify both the true and false branches.
